# 🆕 새로운 기능 추가됨!

## 1️⃣ 지능형 종목 코드 변환 기능

이제 정확한 종목 코드를 몰라도 괜찮습니다! AI가 자동으로 올바른 종목 코드를 찾아줍니다.

### ✨ 주요 기능

**한글 입력 지원:**
- "삼성전자", "SK하이닉스", "카카오" 등 한글로 입력 가능
- 오타도 자동 보정: "삼성전쟈" → "삼성전자 (005930.KS)" ✅

**영어 입력 지원:**
- "Apple", "Tesla", "Microsoft" 등 영어로 입력 가능
- 한글도 가능: "애플" → "Apple (AAPL)" ✅

**자동 변환:**
- GPT-4o-mini를 사용하여 지능적으로 종목 추론
- API 키가 없어도 기본 매칭 기능 제공

### 📍 사용 위치
- **종목 분석 탭**: 종목 코드 입력창
- **포트폴리오 탭**: 종목 추가 입력창

## 2️⃣ AI 투자 어드바이저 챗봇

실시간으로 AI와 대화하며 투자 조언을 받을 수 있습니다!

### 💬 챗봇이 할 수 있는 것

1. **종목 분석 및 투자 조언**
   - "삼성전자 어때?" → 실시간 주가, 뉴스, 감성 분석 제공
   - "테슬라 분석해줘" → 상세한 분석 결과와 투자 의견

2. **포트폴리오 구성 및 관리**
   - "초보 투자자를 위한 포트폴리오 추천해줘"
   - "위험 분산 전략에 대해 알려줘"

3. **투자 전략 및 리스크 관리**
   - "장기 투자 전략이 뭐야?"
   - "안전한 투자 방법 알려줘"

4. **시장 동향 및 트렌드**
   - "요즘 반도체 시장 어때?"
   - "AI 관련 주식 추천해줘"

5. **투자 용어 및 개념 설명**
   - "PER이 뭐야?"
   - "배당수익률 설명해줘"

### 🎯 특징

- **맞춤형 조언**: 사용자의 투자 성향(안정형/중립형/공격형)을 고려한 답변
- **실시간 분석**: 질문에 종목명이 포함되면 자동으로 실시간 분석 수행
- **대화 기록**: 이전 대화 내용을 기억하여 연속된 대화 가능
- **친근한 인터페이스**: 이모지와 함께 읽기 쉬운 답변 제공

### 📝 예시 대화

```
👤 사용자: "삼성전자 분석해주고 지금 사는게 좋을까?"

🤖 FinGenie:
📊 삼성전자 (005930.KS) 분석 결과

현재 주가 정보:
- 현재가: ₩71,500
- 변동률: +2.15%
- 최고가: ₩72,000
- 최저가: ₩69,800

시장 감성: 긍정적 (점수: 65/100)
위험도: 🟡 중간

분석 의견:
삼성전자는 현재 긍정적인 뉴스 흐름을 보이고 있으며...
[상세한 AI 분석 및 투자 조언 계속]
```

### 🚀 사용 방법

1. **AI 챗봇 탭**으로 이동
2. 하단 입력창에 질문 입력
3. 또는 **예시 질문 버튼** 클릭
4. AI가 실시간으로 답변 제공!

## 🔧 기술 스택

- **GPT-4o-mini**: 지능형 종목 코드 변환 및 챗봇 대화
- **LangChain**: AI 에이전트 프레임워크
- **Streamlit**: 웹 인터페이스
- **yFinance**: 실시간 주가 데이터

## ⚙️ 설정 방법

### 필수: OpenAI API 키 설정

`.env` 파일에 API 키를 추가하세요:

```env
OPENAI_API_KEY=your_actual_api_key_here
```

### API 키가 없는 경우

- **종목 코드 변환**: 기본 매칭 기능만 사용 (인기 종목 위주)
- **AI 챗봇**: 사용 불가 (API 키 필요)
