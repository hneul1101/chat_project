# FinGenie 실행 가이드

## 🚀 빠른 시작

### 1. 터미널에서 프로젝트 폴더로 이동
```powershell
cd "c:\Users\214\Desktop\gpgp\chat_project\project"
```

### 2. Streamlit 앱 실행
```powershell
streamlit run app.py
```

### 3. 브라우저에서 자동으로 열립니다!
- 기본 주소: http://localhost:8501

---

## ⚙️ 설정 확인

### OpenAI API 키 설정 (필수!)

`.env` 파일을 열어서 API 키를 입력하세요:

```env
OPENAI_API_KEY=sk-your-actual-api-key-here
```

**API 키가 없으면:**
- ✅ 종목 분석: 정상 작동 (기본 기능)
- ❌ AI 챗봇: 사용 불가
- ⚠️ 종목 코드 변환: 제한적 (기본 매칭만)

---

## 🎯 새로운 기능 사용해보기

### 1️⃣ 지능형 종목 코드 변환
1. **종목 분석** 탭으로 이동
2. 입력창에 "삼성전쟈" 또는 "애플" 입력
3. **분석하기** 버튼 클릭
4. AI가 자동으로 올바른 종목 찾기! ✨

### 2️⃣ AI 챗봇 사용
1. **AI 챗봇** 탭으로 이동
2. 질문 입력 또는 예시 버튼 클릭
3. AI와 대화하며 투자 조언 받기! 💬

---

## 🐛 문제 해결

### 앱이 실행되지 않을 때
```powershell
# 필요한 패키지 재설치
pip install -r requirements.txt
```

### 종목 데이터가 안 나올 때
- 인터넷 연결 확인
- 종목 코드가 올바른지 확인
- 시장 개장 시간인지 확인

### AI 챗봇이 작동하지 않을 때
- `.env` 파일에 API 키가 올바르게 설정되었는지 확인
- API 키 형식: `OPENAI_API_KEY=sk-...`

## 💡 사용 팁

1. **사이드바 설정**
   - 투자 성향을 자신에게 맞게 변경하세요
   - 인기 종목 버튼을 클릭하면 바로 분석 가능!

2. **종목 코드 입력**
   - 한글, 영어 모두 가능합니다
   - "삼성", "애플", "테슬라" 등으로 검색하세요

3. **챗봇 활용**
   - 구체적으로 질문할수록 좋은 답변을 받습니다
   - 예: "삼성전자 분석하고 지금 매수 타이밍인지 알려줘"
