### 🔷 슬라이드 8: 핵심 기술 1 - LangChain & LangGraph (5점)

**제목**: LangChain & LangGraph 활용

```python
# workflow.py - LangGraph 워크플로우 예시

workflow = StateGraph(InvestmentState)

# 노드 정의
workflow.add_node("fetch_data", fetch_stock_data)    # 주가 수집
workflow.add_node("summarize", summarize_news)        # 뉴스 요약
workflow.add_node("sentiment", analyze_sentiment)     # 감성 분석
workflow.add_node("advice", generate_investment_advice)  # 조언 생성

# 워크플로우 연결
workflow.set_entry_point("fetch_data")
workflow.add_edge("fetch_data", "summarize")
workflow.add_edge("summarize", "sentiment")
workflow.add_edge("sentiment", "advice")
```

**시각화**: 워크플로우 다이어그램 이미지 첨부

```
[fetch_data] ──→ [summarize] ──→ [sentiment] ──→ [advice] ──→ [END]
     │               │               │               │
     ▼               ▼               ▼               ▼
  주가/뉴스       LLM 요약       긍정/부정        맞춤 조언
   수집            생성           분석            생성
```
