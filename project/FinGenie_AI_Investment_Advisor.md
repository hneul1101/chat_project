
# AI 투자 어드바이저 챗봇 (FinGenie)

> 생성형 AI와 실시간 검색·분석 기능을 활용한 ‘개인 맞춤형 투자 분석 비서’

---

## 🎯 핵심 아이디어

- 사용자의 투자 성향(예: 위험선호형, 장기투자형 등)을 파악하고  
  뉴스 + RAG 기반 주가 데이터로 **요약 및 리스크 평가** 제공.
- 예: “삼성전자 최근 1개월 이슈 요약해줘” → 뉴스 검색 후 요약 + 긍정/부정 분석  
- 예: “내 포트폴리오 중 위험도가 높은 종목은?” → 데이터 기반 리스크 점수화  

---

## 💡 기술 구성 (Ⅱ항목 고득점 요소)

| 항목 | 구현 아이디어 |
|------|----------------|
| **LangChain & LangGraph** | 뉴스요약 → 감성분석 → 투자조언 순으로 그래프 워크플로 구성 |
| **RAG + 인터넷 검색** | Naver Finance / Yahoo Finance 등 최신 기사 및 주가 검색 |
| **Tools (2개 이상)** | `get_stock_summary()`, `get_sentiment_analysis()` 등 구현 |
| **UI/UX** | Streamlit 기반 대시보드형 챗봇 (차트, 종목 검색창, 추천 카드) |
| **데이터 중심성** | 사용자 입력 종목·기간에 따라 다른 분석 결과 제공 |
| **창의성** | “투자 일기 모드” 추가 → 대화 내용을 요약해 PDF로 내보내기 |

---

## 🎤 발표 전략 (Ⅰ항목 고득점 팁)

| 항목 | 제안 |
|------|------|
| **발표 구성** | 문제 → 해결 아이디어 → 구조/데모 순으로 스토리텔링 |
| **상업성 강조** | “젊은 투자자 대상 AI 재테크 플랫폼으로, 구독형 모델을 통해 수익화 가능” |
| **사용자 니즈 분석** | 대학생 투자자 페르소나 (예: ‘현승, 26세, 첫 투자 1년차’) |
| **시연 완성도** | 실시간 입력 예시 보여주기 (Streamlit 앱 화면 공유) |
| **전달력 및 협업** | 팀 역할 구분 명확히 (예: Front, RAG, LangGraph 담당) |

---

## ✅ 요약

FinGenie는 생성형 AI와 RAG 기반의 뉴스 분석을 결합해,  
사용자 맞춤형 투자 조언과 리스크 평가를 제공하는 **스마트 투자 비서 챗봇**입니다.  
Streamlit 기반으로 시각화된 결과를 제공하여 **상업성, 완성도, 기술성**을 모두 만족시킵니다.
